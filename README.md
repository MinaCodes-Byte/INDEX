## 📋 Project Overview
The **Eid Reward Distribution System (ERDS)** is a secure, encrypted portal designed to manage and distribute digital Eidi packets to verified family members. This system utilizes a 7-stage data synchronization protocol to ensure packet integrity before unlocking the vault.

## 🚀 Features
* **Biometric Verification**: Real-time camera feed integration for retinal and facial scanning.
* **Packet Synchronization**: Multi-stage "Sync" process to verify user data.
* **Dynamic Leaderboard**: Real-time tracking of "Eidi Tax Evaders" and high-priority targets.
* **Auto-Recovery Protocol**: Built-in system fail-safe that triggers during critical process interrupts.

## 🛠️ Installation & Deployment
1.  **Clone the Repository**: Ensure all assets (images and HTML) are in the same directory.
2.  **Asset Check**: Verify that `Favicon.png` and all recipient images (`HIJAB.jpeg`, `ABDULLAH.jpeg`, etc.) are present.
3.  **Host on GitHub Pages**:
    * Go to **Settings > Pages**.
    * Select the `main` branch and `/root` folder.
    * Save to generate your secure live link.

## ⚠️ System Requirements
* **Camera Access**: Required for the biometric "Face-ID" scan.
* **Audio**: Enabled for system status alerts and "verification" sounds.
* **Proximity**: For best results, users should lean close to the screen during the "System Recovery" phase.
